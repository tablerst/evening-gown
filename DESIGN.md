# 晚礼服官网 UI 设计规范（UI Guide）

> 关键词：暗夜、祖母绿、丝绒感、克制、仪式感

本规范用于指导晚礼服官网的整体信息架构、版式、组件与动效设计。所有视觉实现需在本 UI 规范与《Style 色彩规范》约束下完成。

---

## 1. 信息架构与页面结构

### 1.1 顶层结构

- 首页（Home）
  - Hero 首屏：品牌叙事 + 流动丝绸视觉
  - 精选系列（Featured Collection）
  - 热门单品（Best Sellers）
  - 定制服务（Couture & Custom）
  - 品牌故事 / 工坊（Atelier）
- 系列列表（Collections）
- 单品详情（Product Detail）
- 预约试穿 / 定制表单
- 关于品牌 / 联系方式

### 1.2 布局网格

- 桌面端：12 列栅格，内容最大宽度 `1440px`，主体内容居中。
- 内容安全区：左右各保留 `80–120px` 的安全边距，避免文字贴边。
- 组件之间保持充足留白（行距 72–120px），让“空间的浪费”成为高级感的一部分。

---

## 2. Hero 首屏设计（重点）

### 2.1 视觉目标

- 在 0.5 秒内建立品牌记忆：**黑曜石背景 + 祖母绿氛围 + 流光丝绸**。
- 用户情绪：安静、被邀请进入、类似进剧场前的暗厅。

### 2.2 结构分区

Hero 区域高度：至少一屏（100vh），在高分辨率屏幕上允许略超出一屏制造“沉浸感”。

自左至右分为三块：

1. **左侧安全区（品牌与文案）**
   - 左侧内边距：`min(12vw, 180px)`
   - 元素自上而下：
     - 顶部小字：`THE COLLECTION` / `THE EMERALD NIGHT`
     - 主标题：中英文主题词（如 “OBSIDIAN DREAMS”）
     - 副标题：一行简短描述（约 40–60 字以内）
     - 主 CTA：`探索系列`；次 CTA：`预约试穿`
2. **中间过渡区**
   - 主要承载丝绸光带 / 模特主体的过渡区域。
   - 保持文字与丝绸高光之间的视觉距离，避免干扰可读性。
3. **右侧氛围区**
   - 大面积黑曜石 + 祖母绿渐变 + 流动丝绸体积光。
   - 可容纳一部分模型 / 3D 丝绸，允许略超出画面（溢出可视区）。

### 2.3 流动丝绸动效规范

**形态：**

- 只模拟 **一条丝绸 / 布带**，而不是多条光带乱飞。
- 丝绸位置：以右侧 1/3 画面为主，从右上向左下缓慢垂落，形成 S 形弯曲。
- 颜色：
  - 基色：`--brand-emerald` 向 `--bg-obsidian` 过渡，形成深祖母绿。
  - 高光：可偏向香槟色 / 冷银（接近 `--text-champagne` 与 `--accent-silver`）。

**光影与节奏：**

- 动画节奏：单次完整“呼吸”周期 12–20 秒，避免躁动感。
- 高光移动速度远低于游戏特效，接近舞台慢慢变换的追光。
- 悬停无交互要求，只需时间维度变化即可（不做鼠标跟随，避免廉价感）。

**技术实现（建议层面）：**

- 可采用：
  - Three.js / WebGL 布料模拟 + 后期高斯模糊；（暂时选用这个）
  - 或预渲染视频 / Lottie 动画，以 CSS mask / 混合模式叠加。
- 始终保证：
  - 不占用首屏主线程太多性能；
  - 在低性能设备上可自动降级为静态渐变图 + 微弱高光。

### 2.4 文案与排版

- 主标题字体：
  - 英文推荐：Didot / Playfair Display / Cormorant Garamond 等高对比衬线体。
  - 中文可搭配：思源宋体 / 仿宋类高对比宋体。
- 主标题字号：
  - 桌面端：`72–96px`（英文可使用全大写）。
  - 字间距略增，营造呼吸感。
- 颜色：
  - 主标题：`--text-champagne`
  - 副标题：`--text-platinum`
- 对比度要求：
  - 文本背景区域需使用底部渐变遮罩（黑曜石向透明过渡），确保 WCAG AA 对比度。

### 2.5 CTA 按钮规范

- 主按钮（Primary）：
  - 填充色：渐变 `--brand-emerald` → `--brand-emerald-hover`
  - 文本色：`--text-champagne`
  - 边框：1px `--accent-gold`（不必太亮）
  - Hover：
    - 在按钮表面有一条窄窄的高光从左向右缓慢滑过，模拟丝绸光带。
    - 阴影微弱向四周扩散（不要强烈的投影）。
- 次按钮（Secondary）：
  - 透明背景 + 1px `--accent-gold` 边框。
  - 文本色：`--text-champagne`
  - Hover 仅增强边框亮度，不改变填充。

## 3. 其他关键版块 UI 规范

### 3.1 导航栏（Navbar）

- 位置：固定顶部，玻璃拟态样式。
- 背景：`backdrop-filter: var(--glass-blur)`，底色 `rgba(5,5,5,0.7)`
- Logo：
  - 仅使用单色古董金 / 银色线稿版。
  - 不使用大面积金色实心块。
- 文本：
  - 颜色：默认 `--text-platinum`，Hover 使用 `--text-champagne`。
  - 字号：`14–16px`，字间距略加大。
- 滚动行为：
  - 顶部时透明度较低；
  - 向下滚动后轻微缩小高度、加深背景不透明度。

### 3.2 产品卡片（Product Card）

- 背景：
  - 默认透明或 `--bg-surface`（极深森林绿）。
- 图片比例：
  - 建议 3:4 或 4:5，突出垂直感与裙摆。
- 信息层级：
  - 品名：`--text-platinum`，衬线体 / 无衬线均可，但保持低调。
  - 系列名 / 类别：更小字号，`#999999` 左右的灰色。
  - 价格：`--text-champagne`，与主按钮同色系。
- Hover 效果：
  - 图片轻微放大（1.03 倍以内），并添加内发光（内边缘微弱金色/银色光圈）。
  - 文本颜色略微提亮即可，避免夸张动画。

### 3.3 商品详情页（PDP）

- 首屏折叠布局：
  - 左侧：大图 + 缩略图列。
  - 右侧：商品名称、系列名、价格、尺码选择、按钮。
- 主要按钮（加入购物车 / 预约试穿）样式与 Hero CTA 保持一致。
- 详情内容：
  - 面料展示区域允许叠加局部丝绒 / 丝绸放大纹理背景。
  - 规格与说明使用 `--text-platinum`，标题使用 `--text-champagne`。

## 4. 字体与排版体系

### 4.1 字体层级

- H1：Hero 主标题
- H2：版块标题（如“精选系列”）
- H3：卡片 / 子模块标题
- Body：正文文本
- Caption：辅助信息、标签

示例（可根据实际字体调整）：

- H1：72–96px，衬线体，`--text-champagne`
- H2：36–48px，衬线体，`--text-champagne`
- H3：20–24px，无衬线体，`--text-platinum`
- Body：16px，无衬线体，`--text-platinum`
- Caption：12–14px，无衬线体，灰度略低

### 4.2 行高与间距

- 标题行高：1.1–1.2
- 正文行高：1.6–1.8
- 模块内垂直间距：24–40px
- 模块之间：72–120px

---

## 5. 动效规范与降级策略

### 5.1 动效原则

- 所有动效围绕“材质”展开：丝绒的暗涌、丝绸的流光、玻璃的朦胧。
- 禁止：
  - 夸张的弹跳、旋转、强烈缩放。
  - 频闪、强烈发光、颜色骤变。

### 5.2 降级策略

- 在设备性能不足或网络较差时：
  - Hero 丝绸改为静态渐变背景 + 轻微透明高光图层。
  - 去掉复杂粒子 / 3D 效果，仅保留基本渐变与阴影。
- 所有核心信息（标题、按钮）必须在无动效情况下依然清晰易读。

---

## 6. 响应式与移动端适配

- 移动端 Hero 高度仍保持 100vh。
- 丝绸可简化为偏抽象的亮斑 / 曲线，而非具象布料。
- 导航收缩为汉堡菜单，保持玻璃模糊背景。
- 触控态 Hover 效果转化为：
  - 点击轻微高亮；
  - 按钮按下时添加短暂的“亮带扫过”动画。

---

## 7. 无障碍与可用性

- 所有功能性文本颜色组合需满足 WCAG AA 对比度。
- 禁止用金色作为正文文字主色。
- 按钮与链接必须同时通过颜色与形状/下划线区分状态（Hover / Active）。
- 所有动画提供 `prefers-reduced-motion` 支持：在系统设置为减少动效时，关闭丝绸动画，仅保留静态背景。
